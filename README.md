# Outlit Agent Skills

Agent Skills for integrating [Outlit](https://outlit.ai) customer journey analytics.

## Installation

```sh
npx add-skill outlit/outlit-agent-skills
```

## Available Skills

| Skill | Description |
|-------|-------------|
| outlit-integration | Complete SDK integration for React, Next.js, Vue, Node.js |

## Usage

Once installed, agents automatically use these skills when working on Outlit-related tasks.
